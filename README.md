# tri-demo: Sovereign Asset Core
**Protocol:** CRA v2.1 | **Audit Status:** EXTERNAL | **Asset:** GLX-USD

Core repository for the management and settlement of sovereign digital assets on the GlobalLink network. Optimized for hyper-computer execution and 100% DQFR.

* **Consensus:** Hybrid PoS/DPoS
* **Monitoring:** CRA_Watchdog_v1.py
* **Integrity:** 100% Direct Query Fulfillment Rate (DQFR)
